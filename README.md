# JavaScript Exercises
Bu repo kendim için oluşturduğum bir 'JavaScript Rehberi'dir. Yüksek çoğunluğu [W3Schools](https://www.w3schools.com/js/default.asp) sitesinden alınmış bilgiler olup sitede yayınlı JavaScript eğitiminin Türkçe çevirisi denebilir. Ancak birebir çeviri değildir. Ekleme yaptığım yerler ve örnekler mevcuttur. Konu başlıkları bağlamdan kopmamak adına bilinçli şekilde 'İngilizce' ele alınmıştır. Terimlerde aynı şekilde (çoğu yerde) 'İngilizce' olarak ele alınmıştır. Güncel tutulmaya özen gösterilecek olup araştırılmamış ve doğruluğu teyit edilmemiş hiçbir bilgi içermeyecektir.
